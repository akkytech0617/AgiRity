# Task Completion Checklist

1. **Verification**: Ensure the feature or fix works as expected.
2. **Tests**: Run `npm run test` to ensure no regressions. If new logic was added, add unit tests.
3. **Linting**: Run `npm run lint:fix` to format code and fix style issues.
4. **Type Check**: Run `npm run type-check` to ensure no type errors.
5. **Documentation**: Update relevant documentation if architectural changes were made.
