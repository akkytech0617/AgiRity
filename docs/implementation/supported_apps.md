# Supported Apps - 対応アプリ一覧

## 概要

AgiRityのLauncherServiceで動作確認済みのアプリと、その動作パターンを記録する。

## 動作パターン分類

| パターン               | 説明                                           | `open -a App folder`の動作 |
| ---------------------- | ---------------------------------------------- | -------------------------- |
| **A: フォルダ対応**    | フォルダを引数で受け付ける                     | OK                         |
| **B: AppleScript必要** | `open -a`ではフォルダ指定不可、AppleScript必要 | NG                         |
| **C: URLのみ**         | URLを引数で受け付ける                          | N/A                        |
| **D: 引数不要**        | シングルトンアプリ、引数なしで起動             | N/A                        |
| **E: 特殊対応**        | 専用CLIやカスタム処理が必要                    | 要調査                     |

## 未対応・制限事項

- **ターミナル系アプリ**: `open -a`ではフォルダでcdされない。AppleScript対応が必要（AKK-93）
- **特殊引数が必要なアプリ**: 個別対応またはカスタムコマンド機能で対応予定
